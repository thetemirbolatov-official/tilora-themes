## 🚀 Особенности

### 🎨 Премиум дизайн
- **Космические градиенты** - плавные переходы между фиолетовым (#7B61FF) и синим (#00D4FF)
- **Неоновое свечение** - акцентные элементы с эффектом свечения для фокуса
- **Минималистичный интерфейс** - чистый, современный дизайн без лишних элементов
- **Семантическое выделение** - цветовая кодировка по типу элементов кода

### ⚡ Производительность
- **Совместимость с VS Code 1.60+** - работает даже на старых версиях
- **Низкое потребление ресурсов** - оптимизировано для слабых компьютеров
- **Быстрая загрузка** - моментальная активация темы
- **Адаптивные анимации** - можно отключить для максимальной производительности

### 🌌 Две темы в одном пакете
1. **TILORA Cosmic** - темная космическая тема для ночного кодирования
2. **TILORA Cosmic Light** - светлая версия для дневной работы

### 🔧 Кастомизация
- **5 готовых пресетов** - быстрая смена цветовой схемы
- **Настраиваемые анимации** - контроль скорости и эффектов
- **Кастомные цвета** - возможность задать свой акцентный цвет
- **Адаптивный UI** - элементы интерфейса подстраиваются под тему

## ⚡ Установка

### Способ 1: Через VS Code Marketplace (рекомендуется)
1. Откройте **Extensions Marketplace** в VS Code (`Ctrl+Shift+X`)
2. Найдите **"TILORA THEMES"** в поиске
3. Нажмите кнопку **"Install"**
4. После установки нажмите **"Reload"**

### Способ 2: Из VSIX файла
1. Скачайте файл `.vsix` с [GitHub Releases](https://github.com/thetemirbolatov-official/tilora-themes/releases)
2. В VS Code откройте командную палитру (`Ctrl+Shift+P`)
3. Введите `Extensions: Install from VSIX`
4. Выберите скачанный файл

### Способ 3: Через командную строку
```bash
code --install-extension thetemirbolatov.tilora-themes
```

### Активация темы
После установки:
1. Откройте командную палитру (`Ctrl+Shift+P`)
2. Введите `Preferences: Color Theme`
3. Выберите **"TILORA Cosmic"** или **"TILORA Cosmic Light"**

## ⚙️ Настройки

### Открытие настроек
```json
// settings.json
{
    // Настройки TILORA Themes
    "tiloraThemes.enableAnimations": true,
    "tiloraThemes.animationSpeed": "medium",
    "tiloraThemes.glowEffect": true,
    "tiloraThemes.particleEffects": false,
    "tiloraThemes.customAccentColor": "#7B61FF",
    "tiloraThemes.showWelcome": true
}
```

### Опции настройки
| Настройка | Тип | По умолчанию | Описание |
|-----------|-----|--------------|----------|
| **enableAnimations** | boolean | `true` | Включить плавные анимации |
| **animationSpeed** | string | `"medium"` | Скорость анимаций (slow/medium/fast) |
| **glowEffect** | boolean | `true` | Эффект свечения элементов |
| **particleEffects** | boolean | `false` | Космические частицы (только для мощных ПК) |
| **customAccentColor** | string | `"#7B61FF"` | Кастомный цвет акцента |
| **showWelcome** | boolean | `true` | Показывать приветственное сообщение |

### Оптимизация для разных ПК

#### Для слабых компьютеров (рекомендуется)
```json
{
    "tiloraThemes.enableAnimations": false,
    "tiloraThemes.glowEffect": false,
    "tiloraThemes.particleEffects": false
}
```
**Результат:** Максимальная производительность, минимальное потребление ресурсов.

#### Для средних компьютеров
```json
{
    "tiloraThemes.enableAnimations": true,
    "tiloraThemes.animationSpeed": "fast",
    "tiloraThemes.glowEffect": true,
    "tiloraThemes.particleEffects": false
}
```
**Результат:** Баланс между красотой и производительностью.

#### Для мощных ПК
```json
{
    "tiloraThemes.enableAnimations": true,
    "tiloraThemes.animationSpeed": "medium",
    "tiloraThemes.glowEffect": true,
    "tiloraThemes.particleEffects": true
}
```
**Результат:** Полный визуальный опыт со всеми эффектами.

## 🔄 Обновления

### Как обновлять
VS Code автоматически обновляет расширения. Чтобы проверить обновления вручную:
1. Откройте панель расширений (`Ctrl+Shift+X`)
2. Нажмите на значок обновления в правом верхнем углу
3. Найдите TILORA Themes и обновите если доступно

## ❓ Часто задаваемые вопросы

### ❓ Тема не применяется
1. Перезагрузите VS Code (`Ctrl+Shift+P` → `Developer: Reload Window`)
2. Проверьте, что выбрана правильная тема (`Ctrl+K Ctrl+T`)
3. Проверьте конфликты с другими расширениями

### ❓ Анимации тормозят
1. Отключите анимации в настройках
2. Установите скорость анимаций на "fast"
3. Отключите particleEffects если включены

### ❓ Не работает подсветка для определенного языка
1. Убедитесь, что установлен соответствующий language server
2. Проверьте актуальность расширения языка
3. Сообщите о проблеме на GitHub

### ❓ Как изменить цвет акцента вручную
```json
{
    "tiloraThemes.customAccentColor": "#ВАШ_ЦВЕТ"
}
```
Используйте HEX формат цвета.
